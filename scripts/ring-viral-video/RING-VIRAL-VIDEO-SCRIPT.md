## xAI Grok Video — draft vs production

| Mode | Model | Resolution | ~Cost |
|------|--------|------------|-------|
| **draft** (default CLI) | `grok-imagine-video` | 480p | $0.05/s |
| **production** | `grok-imagine-video` | 720p | $0.05/s |
| **remaster** | `--remaster` → production row in manifest | 720p | links `remasterFromRequestId` |

```bash
# Draft all clips (cheap)
node scripts/ring-viral-video/generate-xai-videos.mjs --draft --download

# Remaster one clip to 720p
node scripts/ring-viral-video/generate-xai-videos.mjs --remaster 01_hoa_video_call_pains --download
```

Manifest is **cumulative** (`id::qualityMode` keys) — partial runs no longer wipe prior clips.

See [Generative videos](/en/docs/development/generative-videos) and MCP `ring-video-create`.
