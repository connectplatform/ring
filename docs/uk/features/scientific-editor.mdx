# Науковий редактор

У v1.6.0 Ring Platform надає редактор наукових публікацій для спільнот із довгими структурованими матеріалами.

## Компоненти

| Шлях | Роль |
|------|------|
| `components/editor/scientific-editor.tsx` | Основний редактор |
| `components/editor/equation-editor.tsx` | Рівняння LaTeX |
| `components/editor/version-history-panel.tsx` | Версії документа |
| `app/(authenticated)/[locale]/editor/page.tsx` | Маршрут редактора |

## API публікацій

- `GET/POST /api/publications`
- `GET/PATCH/DELETE /api/publications/[id]`
- `GET /api/publications/[id]/versions`

## Інтернаціоналізація

Тексти UI в `locales/{en,uk,ru}/editor.json`, завантаження через `next-intl`. Див. [Система локалей](/docs/features/locale-system).
