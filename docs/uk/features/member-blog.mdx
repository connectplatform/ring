# Блоги учасників

v1.6.0 виправляє маршрутизацію блогів і зв’язує їх із пайплайном промоції News Kingdom.

## Формат URL

```
/blog/{username}
/blog/{username}/{slug}
```

Допоміжники: `lib/blog/blog-path.ts` — не хардкодьте шляхи вручну.

## Інтеграція з новинами

Пости учасників можна промотувати на головну стрічку через PaymentConductor і Telegram-затвердження. Див. [News Kingdom](/docs/architecture/news-kingdom).

## Обране

`POST /api/user/favorites` та хук `useContentFavorite`.
