### Step 2: Environment Setup
Copy environment template Generate encryption key

<Code language="bash" title="terminal">

</Code>

### Step 3: Configure Environment
.env.local Google OAuth (get from Google Cloud Console) Firebase (get from Firebase Console)

<Code language="bash" title="terminal">
{`NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=your-nextauth-secret-here
WALLET_ENCRYPTION_KEY=your-generated-key-here

AUTH_GOOGLE_SECRET=your-google-client-secret

<Code language="bash" title="terminal">

## üîß Key Files Explained

### Authentication Config
// auth.ts

<Code language="typescript" title="TypeScript">
{`import NextAuth from "next-auth"
import Google from "next-auth/providers/google"

export const { handlers, auth, signIn, signOut } = NextAuth({
  providers: [Google],
  callbacks: {
    async session({ session, token }) {
      // Add user ID to session
      session.user.id = token.sub
      return session
    }
  }
})`}
</Code>

### Database Configuration
// lib/firebase.ts

<Code language="typescript" title="TypeScript">
{`import { initializeApp, getApps } from 'firebase/app'
import { getFirestore } from 'firebase/firestore'

const firebaseConfig = {
  projectId: process.env.AUTH_FIREBASE_PROJECT_ID,
  // ... other config
}

const app = getApps().length === 0 ? initializeApp(firebaseConfig) : getApps()[0]
export const db = getFirestore(app)`}
</Code>

### API Route Example
// app/api/entities/route.ts

<Code language="typescript" title="TypeScript">
{`import { auth } from '@/auth'
import { NextRequest } from 'next/server'

export async function POST(request: NextRequest) {
  const session = await auth()
  
  if (!session) {
    return Response.json({ error: 'Unauthorized' }, { status: 401 })
  }

  const data = await request.json()
  
  // Create entity in database
  const entity = await createEntity({
    ...data,
    userId: session.user.id
  })

  return Response.json({ entity })
}`}
</Code>

## üö® Common Issues & Solutions

### Issue: "FETCH_FAILED" Error
**Solution**: Check Firebase configuration
Verify project ID Check service account key format Should show: -----BEGIN PRIVATE KEY-----

<Code language="bash" title="terminal">
{`echo $AUTH_FIREBASE_PROJECT_ID

echo $AUTH_FIREBASE_PRIVATE_KEY | head -1`}
</Code>

### Issue: Google OAuth Not Working
**Solution**: Verify OAuth configuration
1. Check Google Cloud Console OAuth settings
2. Add `http://localhost:3000` to authorized origins
3. Add `http://localhost:3000/api/auth/callback/google` to redirect URIs

### Issue: Database Permission Denied
**Solution**: Update Firestore rules
// Firestore Rules

<Code language="javascript" title="JavaScript">
{`rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}`}
</Code>
