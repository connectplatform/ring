# Ring CLI — только enterprise

**Ring CLI** (команда `ring`) **не публикуется** в открытом репозитории GitHub начиная с **v1.6.0**.

## Для сообщества разработчиков

Используйте англоязычную документацию [Community tooling](/docs/library/development/community-tooling) (EN canonical):

- `./install.sh`
- `./scripts/setup-dev.sh`

Полный перевод RU — в [LOCALE-GAPS.md](/docs/content/LOCALE-GAPS.md).

## Ringdom settlers

Управляемый деплой: [ringdom.org](https://ringdom.org)
