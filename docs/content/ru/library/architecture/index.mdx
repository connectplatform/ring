## Режимы бэкенда и базы данных

Для работы приложения нужен **`DB_BACKEND_MODE`** (`k8s-postgres-fcm`, `firebase-full` или `supabase-fcm`). В продакшене с приоритетом Postgres Firebase обычно используется для **FCM**, а не как основное хранилище Firestore. Полная матрица, **`DB_HYBRID_MODE`**, Tunnel и Supabase: **[Режимы бэкенда и базы данных](/ru/library/architecture/backend-modes-and-databases)**.
