# Self-hosted развёртывание

Публичный OSS-репозиторий предназначен для self-host через Docker и `install.sh`. **Kubernetes-манифесты и Ring CLI не публикуются.**

## Что есть в OSS

- `install.sh`, `scripts/setup-dev.sh`, `scripts/run-migration.sh`
- `Dockerfile`, `DOCKER_SETUP.md`
- `env.local.template`

## Рекомендуемый стек

1. `./install.sh`
2. PostgreSQL + [миграции](/docs/library/getting-started/migrations)
3. `DB_BACKEND_MODE=k8s-postgres-fcm`
4. `docker compose up` или `npm run build && npm start`

Управляемый хостинг: [ringdom.org](https://ringdom.org).

См. [OSS vs enterprise](/docs/library/development/oss-vs-enterprise).
