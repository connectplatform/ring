# Система локалей

В v1.6.0 конфигурация локалей централизована в **`lib/locale-config.ts`**.

## Переменные окружения

```bash
NEXT_PUBLIC_SUPPORTED_LOCALES=en,uk,ru
NEXT_PUBLIC_DEFAULT_LOCALE=en
```

Коды должны соответствовать папкам в `locales/`.

## Навигация (канонично)

- `Link`, `useRouter`, `usePathname` из `@/i18n/routing`
- Переключение: `persistRingLocalePreference()` + `replaceLocalePath()`
