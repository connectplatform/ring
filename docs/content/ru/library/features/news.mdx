# Модуль Новостей - Цифровой Газетный Опыт

Модуль новостей Ring Platform преобразует управление контентом в профессиональный цифровой газетный опыт, включая автороцентричные рабочие процессы, расширенные инструменты публикации и корпоративную дистрибуцию контента.

## 🎯 Обзор

Модуль новостей предоставляет полную систему управления контентом, которая конкурирует с профессиональными платформами публикации. Построенный с современными веб-технологиями и разработанный для масштабирования, он позволяет сообществам создавать, управлять и распространять высококачественный контент.

## 🏗️ Архитектура

### Схема Базы Данных

Модуль новостей использует PostgreSQL с JSONB для гибкого хранения контента:

```sql
CREATE TABLE news (
    id VARCHAR(255) PRIMARY KEY,
    data JSONB NOT NULL,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
    updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- Индексы производительности
CREATE INDEX idx_news_author_id ON news ((data->>'authorId'));
CREATE INDEX idx_news_category ON news ((data->>'category'));
CREATE INDEX idx_news_status ON news ((data->>'status'));
CREATE INDEX idx_news_published_at ON news ((data->>'publishedAt'));
CREATE INDEX idx_news_locale ON news ((data->>'locale'));
CREATE INDEX idx_news_translation_group ON news ((data->>'translationGroupId'));
```

### Архитектура Сервисов

Все операции новостей используют слой абстракции `DatabaseService` с паттернами React 19:

- **Операции чтения**: `cache()` для оптимизации производительности
- **Мутации**: `revalidatePath()` для свежести UI
- **Транзакции**: Гарантии ACID PostgreSQL для целостности данных

## 📝 Управление Контентом

### Структура Статьи

Каждая новостная статья содержит всесторонние метаданные:

```typescript
interface NewsArticle {
  id: string
  title: string
  slug: string
  content: string // HTML контент
  excerpt: string
  authorId: string
  authorName: string
  category: NewsCategory
  tags: string[]
  featuredImage?: string
  gallery?: string[]
  status: 'draft' | 'published' | 'archived'
  visibility: 'public' | 'subscriber' | 'member' | 'confidential'
  featured: boolean
  views: number
  likes: number
  comments: number
  publishedAt?: Date
  createdAt: Date
  updatedAt: Date
  seo?: NewsSEO
  // Многоязычная поддержка
  locale?: string
  translationGroupId?: string
  availableTranslations?: string[]
}
```

### Категории

Модуль поддерживает 9 предопределенных категорий:

- **platform-updates**: Последние функции и улучшения Ring Platform
- **partnerships**: Сотрудничество и интеграции
- **community**: События сообщества и истории участников
- **industry-news**: Развития Web3 и блокчейна
- **events**: Предстоящие события и собрания
- **announcements**: Важные сообщения платформы
- **press-releases**: Официальные пресс-релизы
- **tutorials**: Руководства и образовательный контент
- **other**: Различный контент

