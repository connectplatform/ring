# API Администрирования

Администрирование платформы с **5 конечными точками** для управления пользователями, администрирования ролей, конфигурации системы и доступа к аналитике.

## Обзор

Система администрирования платформы Ring предоставляет:
- Управление пользователями и назначение ролей
- Конфигурация системы и настройки
- Аналитика платформы и отчетность
- Управление конфигурацией белого лейбла
- Инструменты безопасности и соответствия

## Конечные точки API

### `GET /api/admin/users`
Список и управление пользователями платформы

### `PUT /api/admin/users/{id}/role`
Обновление роли пользователя и разрешений

### `GET /api/admin/analytics`
Получение аналитики и метрик платформы

### `GET /api/admin/config`
Получение конфигурации системы

### `PUT /api/admin/config`
Обновление конфигурации системы

## Возможности Администрирования

- **Управление Пользователями** - Назначение ролей и администрирование пользователей
- **Панель Аналитики** - Метрики использования и производительности платформы
- **Конфигурация Системы** - Переключатели функций и настройки
- **Управление Белым Лейблом** - Элементы управления кастомизацией бренда
- **Мониторинг Безопасности** - Логи доступа и события безопасности

> [!WARNING]
> Конечные точки администрирования требуют роль **ADMIN** и имеют ограничения по частоте запросов для безопасности.
