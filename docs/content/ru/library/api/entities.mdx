# API Сущностей

Управление организациями и бизнесом с **8 конечными точками** для комплексного управления сущностями, включая контроль доступа на основе ролей и отраслевые функции.

## Обзор

Сущности представляют организации, предприятия и профессиональные группы в экосистеме платформы Ring.

**Ключевые Возможности:**
- 26 типов отраслей с визуальными иконками
- Система значков верификации
- Защита конфиденциальных сущностей
- Контроль доступа на основе ролей

## Конечные Точки API

### `GET /api/entities`
Список сущностей с фильтрацией и пагинацией

### `POST /api/entities`
Создание новой сущности (требует роль MEMBER)

### `GET /api/entities/{id}`
Получение деталей сущности по ID

### `PUT /api/entities/{id}`
Обновление информации о сущности

### `DELETE /api/entities/{id}`
Удаление сущности (только для администраторов)

### `POST /api/entities/{id}/verify`
Запрос верификации сущности

### `GET /api/entities/{id}/members`
Список участников сущности и их ролей

### `POST /api/entities/{id}/members`
Добавление участника в сущность

## Структура сущности

```typescript
interface Entity {
  id: string
  name: string
  description: string
  industry: IndustryType
  website?: string
  location?: string
  verified: boolean
  confidential: boolean
  createdAt: Date
  updatedAt: Date
  members: EntityMember[]
}

interface EntityMember {
  userId: string
  role: 'OWNER' | 'ADMIN' | 'MEMBER'
  joinedAt: Date
}
```

## Примеры Реализации

### Create Entity

```typescript
const response = await fetch('/api/entities', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${token}`
  },
  body: JSON.stringify({
    name: 'Tech Innovators Inc',
    description: 'Leading software development company',
    industry: 'TECHNOLOGY_SOFTWARE',
    website: 'https://techinnovators.com',
    location: 'San Francisco, CA'
  })
})

const entity = await response.json()
```

### Список Сущностей с Фильтрацией

```typescript
const response = await fetch('/api/entities?industry=TECHNOLOGY_SOFTWARE&verified=true&limit=10')
const { entities, total, hasMore } = await response.json()
```

## Контроль Доступа

### Видимость Сущностей
- **Публичные Сущности**: Видны всем пользователям
- **Конфиденциальные Сущности**: Требуют роль CONFIDENTIAL или выше

### Управление Сущностями
- **Создание**: Требует роль MEMBER
- **Обновление**: Участники сущности и участники с ролью ADMIN
- **Удаление**: Только участники с ролью ADMIN
- **Верификация**: Специальный процесс верификации

### Роли Участников
- **ВЛАДЕЛЕЦ**: Полный контроль над сущностью
- **АДМИНИСТРАТОР**: Управление участниками и контентом
- **УЧАСТНИК**: Базовый доступ к сущности

## Процесс Верификации

Сущности могут запросить значки верификации:

1. Подача заявки на верификацию
2. Предоставление необходимой документации
3. Процесс проверки платформой
4. Присвоение значка верификации

**Преимущества Верификации:**
- Повышенная достоверность
- Приоритет в результатах поиска
- Доступ к премиум функциям
- Создание конфиденциальных сущностей
