# Предварительные требования

Убедитесь, что у вас есть следующее перед установкой:

- Node.js 22.9+ и npm 10.8+
- Проект Firebase с Firestore + Аутентификацией
- Учетные данные Google Cloud OAuth (Client ID/Secret)
- Возможность безопасно устанавливать переменные окружения
- Доступный OpenSSL для генерации ключей

## Проверить версии

<Code language="bash" title="node -v">
{`npm -v
openssl version`}
</Code>
