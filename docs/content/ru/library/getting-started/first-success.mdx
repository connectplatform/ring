# Первый успех ✅

Используйте этот чек-лист для комплексной проверки вашей настройки.

## Аутентификация

- Перейдите на `/login` и войдите с помощью Google.
- Подтвердите, что сессия существует и профиль загружается.

## Данные

- Откройте `/entities` и убедитесь, что список загружается без ошибок.
- Если пусто, создайте сущность по адресу `/entities/add`.

## Здоровье API

<Code language="bash" title="terminal">
{`curl -I http://localhost:3000/api/health`}
</Code>

Ожидается: HTTP/1.1 200 OK.

## Реальное время

- Запустите уведомление и подтвердите, что оно приходит мгновенно.

## Кошелек

- Посетите профиль; подтвердите, что кошелек существует или создается при первом доступе.
