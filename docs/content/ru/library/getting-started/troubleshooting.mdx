# Устранение неполадок

## Распространенные проблемы

### "FETCH_FAILED" на страницах
- Проверьте учетные данные Firebase Admin SDK в `.env.local`.
- Убедитесь, что Firestore включен.

### Кошелек не найден
- Нормально для новых пользователей; кошелек создается при первом доступе.
- Проверьте логи сервера на ошибки создания.

### Google Sign-in не работает
- Проверьте URI перенаправления OAuth.
- Убедитесь, что провайдер Google включен в Firebase Auth.
