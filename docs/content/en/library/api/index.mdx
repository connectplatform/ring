# API Reference

Welcome to the **Ring Platform API** — **132 route handlers** across entities, opportunities, store, payments (PaymentConductor), news, wallet, tunnel, and more (v1.6.0).

## 🚀 Quick Start

### Base URL
```
https://ring.ck.ua/api
```

### Authentication
All API requests require authentication using Auth.js v5:

<Code language="bash" title="curl -H "Authorization: Bearer YOUR_TOKEN" \">
{`https://ring.ck.ua/api/entities`}
</Code>
