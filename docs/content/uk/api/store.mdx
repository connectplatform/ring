# API магазину

Платформа Ring надає комплексний мульти-вендорний маркетплейс із **12 REST кінцевими точками** для управління продавцями, каталогом продуктів, обробкою замовлень та автоматизованими розрахунками. Система інтегрується з WayForPay для безпечної обробки платежів та підтримує як токени RING, так і традиційні валюти.

<Callout type="info" title="💰 PaymentConductor (v1.6.0)">
Checkout через **PaymentConductor** (`store_order`): WayForPay/Stripe redirect, **внутрішній кредит** (`POST /api/store/payments/credit`). Див. [PaymentConductor](/docs/library/features/payment-conductor).
</Callout>

## 🏗️ Архітектура системи

### Потік магазину

```
Лістинг продукту → Створення замовлення → Обробка платежу → Виконання → Розрахунок
                                      ↓
                            WayForPay API / Переказ RING
```

### Функції мульти-вендора

- **Онбординг продавців**: Процес заявки та верифікації
- **Управління продуктами**: Каталог із відстеженням інвентарю
- **Обробка замовлень**: Автоматизовані робочі процеси виконання
- **Інтеграція платежів**: Двосистема оплати (WayForPay + RING)
- **Автоматизація розрахунків**: Виплати продавцям та збір комісій
- **Аналітична панель**: Відстеження продажів та продуктивності

### Безпека та довіра

- **Верифікація продавців**: Багаторівневий процес верифікації
- **Рейтинг довіри**: Автоматизована система репутації продавця
- **Запобігання шахрайству**: Валідація платежів та захист від чарджбеків
- **Ескроу система**: Безпечне утримання коштів під час транзакцій

## 📋 Довідка кінцевих точок API

### `GET /api/store/products`

Список продуктів із розширеним фільтруванням, пошуком та пагінацією.

#### Параметри

| Параметр | Тип | Обов'язковий | Опис |
|-----------|-----|--------------|------|
| `page` | number | Ні | Номер сторінки (за замовчуванням: 1) |
| `limit` | number | Ні | Продуктів на сторінку (за замовчуванням: 20, макс: 100) |
| `search` | string | Ні | Пошук за назвою продукту, описом, тегами |
| `category` | string | Ні | Фільтр за категорією |
| `vendorId` | string | Ні | Фільтр за конкретним продавцем |
| `minPrice` | number | Ні | Мінімальний фільтр ціни |
| `maxPrice` | number | Ні | Максимальний фільтр ціни |
| `currency` | string | Ні | Фільтр за валютою (RING, USD, EUR, UAH) |
| `inStock` | boolean | Ні | Показувати тільки продукти в наявності |
| `sortBy` | string | Ні | Сортувати за: `price`, `createdAt`, `rating`, `popularity` |
| `sortOrder` | string | Ні | Порядок сортування: `asc`, `desc` |
| `featured` | boolean | Ні | Показувати тільки рекомендовані продукти |

#### Приклад запиту

## 💰 Інтеграція платежів

### Інтеграція WayForPay

Платформа Ring інтегрується з WayForPay для безпечної обробки платежів.

#### Підтримувані методи оплати

- **Кредитні/Дебетові картки**: Visa, Mastercard, American Express
- **Цифрові гаманці**: Apple Pay, Google Pay
- **Банківські перекази**: Прямі платежі з банківського рахунку
- **Криптовалюта**: Токени RING через смарт-контракти

#### Потік платежу

// Клієнтська ініціація платежу

<Code language="typescript" title="TypeScript">
{`const initiatePayment = async (orderId: string) => {
  const response = await fetch(`/api/store/orders/${orderId}/payment`, {
    method: 'POST',
    body: JSON.stringify({
      method: 'wayforpay',
      returnUrl: `${window.location.origin}/order/${orderId}/success`,
      cancelUrl: `${window.location.origin}/order/${orderId}/cancel`
    })
  })

  const { paymentUrl } = await response.json()

  // Перенаправлення до WayForPay
  window.location.href = paymentUrl
}`}
