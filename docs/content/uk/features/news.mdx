# Модуль Новин - Цифровий Газетний Досвід

Модуль новин Ring Platform перетворює управління контентом на професійний цифровий газетний досвід, включаючи автороцентричні робочі процеси, розширені інструменти публікування та корпоративну дистрибуцію контенту.

## 🎯 Огляд

Модуль новин забезпечує повну систему управління контентом, яка конкурує з професійними платформами публікування. Побудований із сучасними веб-технологіями та розроблений для масштабування, він дозволяє спільнотам створювати, керувати та розповсюджувати високоякісний контент.

## 🏗️ Архітектура

### Схема Бази Даних

Модуль новин використовує PostgreSQL з JSONB для гнучкого зберігання контенту:

```sql
CREATE TABLE news (
    id VARCHAR(255) PRIMARY KEY,
    data JSONB NOT NULL,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
    updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- Індекси продуктивності
CREATE INDEX idx_news_author_id ON news ((data->>'authorId'));
CREATE INDEX idx_news_category ON news ((data->>'category'));
CREATE INDEX idx_news_status ON news ((data->>'status'));
CREATE INDEX idx_news_published_at ON news ((data->>'publishedAt'));
CREATE INDEX idx_news_locale ON news ((data->>'locale'));
CREATE INDEX idx_news_translation_group ON news ((data->>'translationGroupId'));
```

### Архітектура Сервісів

Всі операції новин використовують шар абстракції `DatabaseService` з патернами React 19:

- **Операції читання**: `cache()` для оптимізації продуктивності
- **Мутації**: `revalidatePath()` для свіжості UI
- **Транзакції**: Гарантії ACID PostgreSQL для цілісності даних

### Структура Статті

Кожна стаття новин містить всебічну метадані:

```typescript
interface NewsArticle {
  id: string
  title: string
  slug: string
  content: string // HTML контент
  excerpt: string
  authorId: string
  authorName: string
  category: NewsCategory
  tags: string[]
  featuredImage?: string
  gallery?: string[]
  status: 'draft' | 'published' | 'archived'
  visibility: 'public' | 'subscriber' | 'member' | 'confidential'
  featured: boolean
  views: number
  likes: number
  comments: number
  publishedAt?: Date
  createdAt: Date
  updatedAt: Date
  seo?: NewsSEO
  // Багатомовна підтримка
  locale?: string
  translationGroupId?: string
  availableTranslations?: string[]

```

### Категорії

Модуль підтримує 9 попередньо визначених категорій:

- **platform-updates**: Останні функції та покращення Ring Platform
- **partnerships**: Співпраці та інтеграції
- **community**: Події спільноти та історії учасників
- **industry-news**: Розвитки Web3 та блокчейну
- **events**: Майбутні події та зібрання
- **announcements**: Важливі повідомлення платформи
- **press-releases**: Офіційні прес-релізи
- **tutorials**: Посібники та навчальний контент
- **other**: Різний контент
