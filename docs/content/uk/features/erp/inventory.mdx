# Склад і залишки

**Warehouse Manager** (`ERPStockService`) веде залишки на складі `zero-warehouse`, списує товар після оплати та логує рухи в `stock_movements`.

> **EN canonical:** [Inventory & Stock (EN)](/docs/library/features/erp/inventory)

## Можливості

- `getStockSummary()` — дашборд (в наявності, low/critical/out, вартість).
- `updateStock()` — add / subtract / set з причиною.
- `getRecentStockMovements()` — аудит з `stock_movements`.
- **Реферальні продажі** — рухи містять `referralCode` і `assisted: true`, коли на замовленні є атрибуція.

`POST /api/erp/stock/initialize` — масова ініціалізація (admin).
