# Інтеграція Web3

Старі приклади на **ethers.js** зняті. Клони Ring використовують **Wagmi v3 + viem** і **власний UI** гаманця.

**Актуальна документація:**

- **[Ethereum гаманці (Wagmi v3)](/docs/integrations/ethereum-wallets)**

Мінімальний клієнтський приклад — як у [англійській версії цієї сторінки](/docs/examples/web3-integration) (блок `useConnection` / `useConnect`).
