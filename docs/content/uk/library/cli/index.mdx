# Ring CLI — лише enterprise

**Ring CLI** (`команда ring`) **не публікується** у відкритому репозиторії GitHub починаючи з **v1.6.0**.

## Для спільноти розробників

Використовуйте англомовну документацію [Community tooling](/docs/library/development/community-tooling) (EN canonical):

- `./install.sh`
- `./scripts/setup-dev.sh`

Повний переклад UK — у [LOCALE-GAPS.md](/docs/content/LOCALE-GAPS.md).
