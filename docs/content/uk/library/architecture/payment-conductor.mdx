# Архітектура PaymentConductor

| Модуль | Відповідальність |
|--------|------------------|
| `payment-conductor.ts` | Оркестрація create → redirect / credit / token |
| `webhook-dispatcher.ts` | Верифікація та маршрутизація webhook PSP |
| `types.ts` | `PaymentPurpose`, `PaymentProcessorId`, `PaymentRail` |
| `payment-transaction-service.ts` | Запис у реєстр |

Процесори: WayForPay, Stripe, internal credit. Конфіг: `lib/payments/payment.config.ts`.

Ідемпотентність через стабільні `orderReference` у `payment_transactions`.

Див. [PaymentConductor](/docs/library/features/payment-conductor).
