# Брендування

## Логотип та Favicon

- Замініть логотип у Navbar через слот логотипу `app/layout.tsx`.
- Встановіть гліф favicon у `<Head faviconGlyph="🔗" />`.

## Кольори та типографіка

- Токени Tailwind CSS v4: налаштуйте у ваших CSS змінних.
- Надавайте перевагу утилітарним класам над користувацьким CSS.

import { Callout } from '@/components/docs/callout'

<Callout type="info">Зберігайте доступність: контраст AA або краще.</Callout>
