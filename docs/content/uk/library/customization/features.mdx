# Перемикачі функцій

Увімкнення/вимкнення областей платформи під час збірки або виконання.

- Функції: сутності, можливості, повідомлення, сповіщення, магазин, гаманець
- Використовуйте env прапорці (наприклад, `NEXT_PUBLIC_FEATURE_WALLET=true`)
- Уникайте мертвого коду: захищайте маршрути та навігацію
- Надавайте перевагу прапорцям часу компіляції для великих розділів
