# Усунення несправностей

## Поширені проблеми

### "FETCH_FAILED" на сторінках
- Перевірте облікові дані Firebase Admin SDK у `.env.local`.
- Переконайтеся, що Firestore увімкнено.

### Гаманець не знайдено
- Нормально для нових користувачів; гаманець створюється при першому доступі.
- Перевірте логи сервера на помилки створення.

### Google Sign-in не працює
- Перевірте URI перенаправлення OAuth.
- Переконайтеся, що провайдер Google увімкнено у Firebase Auth.
