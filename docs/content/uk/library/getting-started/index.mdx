# Початок роботи

Ласкаво просимо до платформи Ring! Цей посібник допоможе вам запустити систему за лічені хвилини.

## Швидкий старт (5 хвилин)

<Steps>
<Step>
### Клонуйте репозиторій
<Code language="bash" title="terminal">
{`git clone https://github.com/connectplatform/ring.git
cd ring`}
</Code>
</Step>
<Step>
### Встановіть залежності
<Code language="bash" title="terminal">
{`npm install`}
</Code>
</Step>
<Step>
### Налаштуйте середовище
cp env.local.template .env.local
<Code language="bash" title="terminal">
{`npm run setup:env`}
</Code>
</Step>
<Step>
### Запустіть сервер розробки
<Code language="bash" title="terminal">
{`npm run dev`}
</Code>
</Step>
</Steps>
