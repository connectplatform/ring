# Environment Variables (Production)

## Локалі (v1.6.0)

```bash
NEXT_PUBLIC_SUPPORTED_LOCALES=en,uk,ru
NEXT_PUBLIC_DEFAULT_LOCALE=en
```

Див. `lib/locale-config.ts` та [Система локалей](/docs/library/features/locale-system).

## PaymentConductor (v1.6.0)

```bash
PAYMENT_DEFAULT_PROCESSOR=wayforpay
PAYMENT_STORE_PROCESSOR=wayforpay
WAYFORPAY_MERCHANT_ACCOUNT=
WAYFORPAY_SECRET_KEY=
STRIPE_SECRET_KEY=
STRIPE_WEBHOOK_SECRET=
```

Див. [PaymentConductor](/docs/library/features/payment-conductor).
