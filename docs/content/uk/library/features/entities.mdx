# Система сутностей

Управління організаціями з **26 типами галузей**, значками верифікації та контролем доступу на основі ролей.

## Огляд

Система сутностей є основою підходу платформи Ring до професійного нетворкінгу. Сутності представляють організації, бізнеси та професійні групи, які створюють можливості та з'єднуються з іншими сутностями.

## Ключові функції

### 26 типів галузей
Кожна сутність класифікується в одну з 26 галузевих категорій:
- Технології та програмне забезпечення
- Охорона здоров'я та медицина
- Фінанси та банківська справа
- Освіта та навчання
- Та ще 22 спеціалізовані галузі

### Система верифікації
- **Значки верифікації** - Підвищена довіра для верифікованих сутностей
- **Процес документування** - Безпечний робочий процес верифікації
- **Преміум-функції** - Доступ до розширених можливостей платформи

### Контроль доступу
- **Публічні сутності** - Видимі для всіх користувачів платформи
- **Конфіденційні сутності** - Обмежений доступ для чутливих організацій
- **Дозволи на основі ролей** - Ієрархія власник, адміністратор, учасник

## Реалізація

```typescript
// Create new entity
const entity = await createEntity({
  name: 'Технологічні Інноватори ТОВ',
  description: 'Провідна компанія з розробки програмного забезпечення',
  industry: 'TECHNOLOGY_SOFTWARE',
  website: 'https://techinnovators.com'
})

// Request verification
await requestVerification(entity.id, {
  documents: ['business-license.pdf', 'tax-certificate.pdf'],
  contactPerson: 'John Doe',
  contactEmail: 'john@techinnovators.com'
})
```
## Управління сутностями

### Процес створення
1. **Базова інформація** - Назва, опис, галузь
2. **Контактні дані** - Веб-сайт, місцезнаходження, контактна інформація
3. **Верифікація** - Опціональний процес верифікації
4. **Запрошення учасників** - Додавання членів команди з ролями

### Ролі учасників
- **ВЛАСНИК** - Повний контроль сутності та налаштувань
- **АДМІНІСТРАТОР** - Управління учасниками та розміщення можливостей
- **УЧАСНИК** - Базовий доступ до сутності та участь
