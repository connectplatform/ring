# Wallet API

Управління Web3 гаманцем з **10 ендпоінтами**, що підтримують мульти-токенові гаманці, транзакції та інтеграцію з EVM.

## Огляд

Система гаманців Ring Platform надає:
- Автоматичне створення гаманців для користувачів
- Підтримка мульти-токенів (RING, DAAR, DAARION)
- Управління транзакціями та історія
- Відстеження та оновлення балансів
- Інтеграція з блокчейном EVM

## API Ендпоінти

### `GET /api/wallet`
Отримати інформацію про гаманець користувача

### `POST /api/wallet/create`
Створити новий гаманець для користувача

### `GET /api/wallet/balance`
Отримати баланси токенів гаманця

### `POST /api/wallet/transfer`
Перевести токени між гаманцями

### `GET /api/wallet/transactions`
Отримати історію транзакцій

### `POST /api/wallet/transactions/{id}/confirm`
Підтвердити очікувану транзакцію

### `GET /api/wallet/tokens`
Список підтримуваних токенів

### `POST /api/wallet/tokens/add`
Додати користувацький токен до гаманця

### `GET /api/wallet/nfts`
Отримати колекцію NFT користувача

### `POST /api/wallet/export`
Експортувати приватний ключ гаманця (безпечно)

## Підтримувані Токени

- **RING** - Утилітарний токен платформи
- **DAAR** - Токен управління
- **DAARION** - Токен винагород за стейкінг
- **ETH** - Нативний токен Ethereum
- **Користувацькі ERC-20** - Токени, додані користувачем
