# OSS vs enterprise

Ring Platform v1.6.0 чітко розділяє **відкритий OSS** і **enterprise-операції Ringdom**.

## Публічний OSS (GitHub)

| Включено | Примітки |
|----------|----------|
| Повний код застосунку | Next.js 16, React 19, усі модулі |
| `install.sh` + community scripts | Setup, env, міграції |
| Документація (`/docs`) | EN canonical, UK/RU у другому проході |
| Env templates | Без реальних секретів |
| Docker | Self-hosted шлях |

| Виключено (gitignore) | Чому |
|-----------------------|------|
| `k8s/` | Прод-топологія + секрети |
| `cli/` | Empire deploy CLI |
| `propagation/` | Маніфести синхронізації клонів |
| Внутрішні ops scripts | `setup-github-secrets`, `sync-*-to-clones` тощо |

## Ringdom enterprise

- Керований Kubernetes у вашому регіоні
- Reggie AI ringization проєктів
- Legiox skillsets (NFT-gated)
- Кастомна propagation і моніторинг

Реєстрація: [ringdom.org/settler](https://ringdom.org/settler)
