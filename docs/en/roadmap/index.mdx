## Documentation visualization phases

Ring docs use **Mermaid**, **Callout**, and **Steps** today. **Code** blocks use unified server Shiki (`highlightCodeToHtml`) — fences and `<Code>` share one path (no client Shiki). Phase 1 adds **Timeline** (react-chrono), **Math** (KaTeX), and **CodeSandbox** (Sandpack) — live on the public roadmap page.
