# Архитектура PaymentConductor

| Модуль | Ответственность |
|--------|-----------------|
| `payment-conductor.ts` | Оркестрация create → redirect / credit / token |
| `webhook-dispatcher.ts` | Верификация и маршрутизация webhook PSP |
| `types.ts` | `PaymentPurpose`, `PaymentProcessorId`, `PaymentRail` |
| `payment-transaction-service.ts` | Запись в реестр |

Процессоры: WayForPay, Stripe, internal credit. Конфиг: `lib/payments/payment.config.ts`.

Идемпотентность через стабильные `orderReference` в `payment_transactions`.

См. [PaymentConductor](/docs/features/payment-conductor).
