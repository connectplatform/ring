# Кастомизация

<Callout type="info">
  **Что это за раздел:** Ring **всегда** white-label — один open-source код, ваш бренд, ваш домен. Эти доки для **операторов и разработчиков**, которые ведут клон (сами или с Ringdom). Это **не** процесс заказа на Ringdom.
</Callout>

## Два способа получить клон

| Шаг | OSS (вы ведёте) | Ringdom (мы ведём) |
|-----|-----------------|---------------------|
| 1 | [Quick Start](/docs/customization/quick-start) — EN canonical | [ringdom.org](https://ringdom.org) — чат на главной |
| 2 | [Customization guide](/docs/customization/customization-guide) — EN canonical | После handoff — тот же guide |
| 3 | [Self-hosted](/docs/deployment/self-hosted) | Ringdom k8s (не Quick Start) |

<Callout type="warning">
  Для Ringdom-managed settlers **не** используйте Quick Start — заказ начинается на [ringdom.org](https://ringdom.org).
</Callout>
