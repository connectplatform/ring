# Склад и остатки

**Warehouse Manager** (`ERPStockService`) ведёт остатки на складе `zero-warehouse`, списывает товар после оплаты и логирует движения в `stock_movements`.

> **EN canonical:** [Inventory & Stock (EN)](/docs/features/erp/inventory)

## Возможности

- `getStockSummary()` — дашборд (в наличии, low/critical/out, стоимость).
- `updateStock()` — add / subtract / set с причиной.
- `getRecentStockMovements()` — аудит из `stock_movements`.
- **Реферальные продажи** — движения содержат `referralCode` и `assisted: true`, когда в заказе есть атрибуция.
