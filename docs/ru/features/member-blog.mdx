# Блоги участников

v1.6.0 исправляет маршрутизацию блогов и связывает их с пайплайном промоции News Kingdom.

## Формат URL

```
/blog/{username}
/blog/{username}/{slug}
```

Хелперы: `lib/blog/blog-path.ts` — не хардкодьте пути вручную.

## Интеграция с новостями

Посты участников можно промотировать на главную ленту через PaymentConductor и Telegram-одобрение. См. [News Kingdom](/docs/architecture/news-kingdom).

## Избранное

`POST /api/user/favorites` и хук `useContentFavorite`.
