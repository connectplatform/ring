Welcome to Ring Platform

Ring Platform is a comprehensive professional networking solution that combines traditional networking features with cutting-edge Web3 integration, white-label capabilities, and advanced opportunity management.

This documentation is powered by Fumadocs v15.7.7 with advanced features including TypeScript integration, OpenAPI support, and multi-language capabilities.

🚀 Quick Start

Prerequisites

Ensure you have Node.js 18.17+ and npm installed on your system.

Installation

Clone the repository and install dependencies:

git clone https://github.com/connectplatform/ring.git
cd ring
npm install

Configuration

Set up your environment variables:

cp .env.local.template .env.local
# Edit .env.local with your configuration

Launch

Start the development server:

npm run dev

Visit http://localhost:3000 to see your Ring Platform instance.

🎯 Core Features

🏢 Entity Management

Comprehensive organization management with industry-specific features, verification systems, and role-based access control.

  • 26 industry types with visual icons
  • Verification badge system
  • Confidential entity protection
  • Role-based access control

💼 Opportunity Marketplace

Unique dual-nature system supporting both offers (what you provide) and requests (what you need) across multiple opportunity types.

  • Job opportunities
  • Partnership opportunities
  • Volunteer opportunities
  • Mentorship programs

💰 Web3 Integration

Built-in wallet management, multi-token support (RING, DAAR, DAARION), and EVM blockchain integration.

  • Automatic wallet creation
  • Multi-token support
  • Transaction management
  • NFT marketplace
